# MarkDown

**基本語法**

- `<syntax>{space}{content}`
- 換行要 2 個空格

**標題**

- '#' > H1
- '##' > H2
- ...依次類推

**quote**

- '> {content}'
  > this is quote

**code**

```
<p> this is code scope </p>
```

**text highlight**

- '\*\* **' **加粗\*\*
- '~{內容}~' ~~刪除我~~
- `標記`

# Js

**基本聲明**
- const => 不變的  所謂的值  
- let => 可變的 所謂的變量
- var => 可變的 全球性的 `盡量不要用`
  
**變量提升**

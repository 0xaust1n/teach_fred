# ID and Class

- Class像是一個群組 Selector 是 .

# 權重
**權重計算順序**

`inline style  > ID > Class > Element > *`

**權重處理順序**

相同權重 : 後者會覆蓋前者

當多個選取器作用在同一個元素上 : 權重高的優先生效
